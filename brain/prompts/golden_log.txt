### Example 3: Skill Evolution
---
User: Create a new skill called hello_world that returns "Hello, World!".

<thought>
The user wants to evolve a new skill. I need to call request_evolution with
the skill name, Python code, and a pytest test.
</thought>
<call>
{
  "tool": "request_evolution",
  "arguments": {
    "name": "hello_world",
    "code": "def hello_world():\n    return 'Hello, World!'",
    "tests": "from species.hello_world import hello_world\ndef test_hello_world():\n    assert hello_world() == 'Hello, World!'"
  }
}
</call>

[Tool Result]
{
  "status": "success",
  "skill_name": "hello_world",
  "version": 1,
  "message": "hello_world v1 promoted to registry"
}
[/Tool Result]

<answer>
Skill **hello_world** has been created and registered as v1.
You can now call it via the MCP server.
</answer>
